일반적으로 하나의 독립변수와 하나의 종속변수

독립변수에 대한 종속변수가 존재할때
관계가 있는지 없는지

- regression

Supervised Learning Type은 크게 3가지
- 1. Linear Regression
	- 독립변수에 따른 종속변수의 크기 변화량
 	- 데이터가 선형으로 표현되어 있음
		 - 해당 데이터를 1차원선으로 표현 가능
- 2. Logistic Regression(Binary Classification)
	- 결과 값이 2분화, 둘중에 하나의 값으로 떨어짐
- 3. Multinomial Classification
	- 결과 값이 범주에 매핑

Linear는 선형
Logistic은 값이 둘 중에 하나로 떨어지고
Multinomial은 여러개의 범주 중에 하나로 떨어짐

입력 데이터(독립변수와 종속변수)의 형태에 따라 어떤 지도학습을 쓸지

Regression Anlysis
- 일반적으로 머신러닝에서 예측모델을 만들기 위해서 가장 많이쓰고 정형화되고 중요한 자료분석 방법
- 여러 검정 방법 중 하나
- 두개의 데이터가 서로 연관관계를 갖고 있냐 없냐를 판단
- 특정 상황이 아니라 독립변수와 종속변수가 있는데 이 두개가 관련이 있는지 없는지를 분석해낼 수 있음
	- 관계가 있다면 예측모델까지 만들어 줌

회귀분석은 크게 두가지가 있음
- 단순 회귀 분석(독립변수가 하나일 경우)
- 다중회귀분석 (독립변수가 둘 이상인 경우)

단순 선형 회귀 모델
독립변수와 종속변수간 관계를 파악하기위해 그 값들을 대표하는 선을 그음
- 이 선을 찾아내는게 목표
- 이 선을 예측모델이라 할 수 있음
